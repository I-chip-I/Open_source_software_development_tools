# Курсовая работа по дисциплине "Инструментальные средства разработки программного обеспечения с открытым исходным кодом"
## Тема: разработка видеоигрового прототипа

### Уже реализовано:

Описание модулей:

game.tscn, player.tscn, enemy.tscn, ground.tscn, hit_box.tscn, hurt_box.tscn, decorative_objects.tscn, inventory.tscn, pause_menu.tscn и т. д. - файлы, описывающие структуру сцен проекта.

project.godot - основной файл проекта, содержащий общие настройки проекта.

Scripts - папка со скриптами для управления игровым персонажем и логикой вражеского персонажа, для обработки состояний обоих персонажей и переключения между ними.

Test Scripts - папка со скриптами для тестирования модулей.

Enemy/Spritesheets enemy и Player/Spritesheets player - папки со спрайт-листами для визуализации различных состояний врага и игрового персонажа.

icon.svg и icon.svg.import - изображение иконки игрового движка Godot.

### Инструкция пользователя (некорректная версия в отчёте также была уточнена).
1)	Убедитесь, что Ваше устройство имеет системные характеристики не ниже: 
- Процессор: x86_64 c поддержкой SSE4.2,
- Видеокарта: графический процессор с поддержкой Direct3D 11 или OpenGL 3.3,
- Оперативная память: 2 ГБ,
- Место на диске: 150 МБ,
- Операционная система: Windows 10 (x64).
2)	Запустите .exe файл видеоигрового прототипа.
3)	Для перемещения игрового персонажа в игровом мире зажмите одну из четырёх клавиш W, A, S и D.
4)	Для подбора интерактивного предмета игровым персонажем подойдите к нему – он автоматически появится в инвентаре игрового персонажа.
5)	Для вызова окна инвентаря нажмите клавишу E. Помните, что вражеские персонажи всё ещё могут атаковать Вас во время изучения инвентаря.
6)	Для перемещения по ячейкам инвентаря используйте клавиши управления курсором.
7)	Для использования предмета, содержащегося в ячейке, нажмите клавишу Enter.
8)	Для закрытия окна инвентаря повторно нажмите клавишу E.
9)	Для вызова окна меню паузы нажмите клавишу Esc.
10)	Для перемещения в меню паузы используйте клавиши управления курсором.
11)	Для выбора желаемой опции в меню паузы нажмите клавишу Enter.
12)	Для выхода из меню паузы повторно нажмите клавишу Esc.
13)	Для отслеживания состояния здоровья персонажа проверяйте количество отображаемых красных сердец в левом верхнем углу экрана.
