# Курсовая работа по дисциплине "Инструментальные средства разработки программного обеспечения с открытым исходным кодом"
## Тема: разработка видеоигрового прототипа

### Уже реализовано (для практической работы №5):

Описание модулей:

game.tscn и player.tscn - файлы, описывающие структуру сцен проекта.

project.godot - основной файл проекта, содержащий общие настройки проекта.

Scripts - папка со скриптами для управления игровым персонажем, обработки его состояний и их смены.

Enemy/Spritesheets enemy и Player/Spritesheets player - папки со спрайт-листами для визуализации различных состояний врага и игрового персонажа.

icon.svg и icon.svg.import - изображение иконки игрового движка Godot.

### Работа видеоигровго прототипа.
При запуске главной сцены, описанной в game.tscn, или сцены, описанной в player.tscn, запускается пустое окно игры. С помощью нажатия клавиш W, A, S и D происходит управление игровым персонажем, которого можно вывести в видимую область. Скрипт player.gd позволяет передвигать персонажа и определять его направление, с помощью данных о котором рассчитывается спрайт, необходимый для визуализации (видимость предыдущего отключается). Скрипты player_state_machine.gd, state.gd описывают функционал для корректной обработки состояния. Скрипты standing_state.gd и walking_state.gd описывают обработку соответствующего состояния при входе в данное состояние, пребывании в нём и выходе из данного состояния (переключение между спрайтами, выбор соответствующей анимации).
